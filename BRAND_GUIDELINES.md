# Brand Guidelines

## DiffeSense Name and Logo Usage

The "DiffeSense" name and logo are identifiers of this project. Please use them appropriately to avoid confusion about the official project.

---

## Permitted Uses

You are welcome to:

- **Use the software freely** under the Apache-2.0 license
- **Fork and modify** the code for any purpose
- **Reference DiffeSense** when describing your project's origins:
  - "Built on DiffeSense"
  - "Based on DiffeSense"
  - "Fork of DiffeSense"
  - "Powered by DiffeSense"

---

## Please Avoid

To prevent confusion with the official project:

- Using "DiffeSense" as the primary name for a fork or derivative work
- Using the logo in a way that suggests official endorsement
- Creating the impression that your fork is the official project

---

## Naming Forks

If you fork DiffeSense, please use a distinct name:

- ✅ "MyRiskTool (based on DiffeSense)"
- ✅ "CodeRiskAnalyzer - fork of DiffeSense"
- ❌ Names that could be confused with the official project

---

## Why This Matters

Clear branding helps:

1. **Users** know they're getting the official, maintained version
2. **Contributors** understand where to submit improvements
3. **Everyone** benefits from a clear ecosystem
